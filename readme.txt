== Description ==

This plugin is used to showcase portfolio along with project link which open in an iframe <br><br> 
**NOTE: Please use shortcode ``[portfolio]`` for showing your portfolio at any page.**

= Main Features For Free =

* simple layout for showcasing portfolio items.
* Add project links which will open in an iframe.

== Installation ==

To install this plugin follow below steps

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add portfolio items from **'Sidebar '** -> **'jk portfolio '** -> **'Add new'** 
4. use shortcode ``[portfolio]`` for showing your portfolio at any page.
5. That's it, no other settings needed.
